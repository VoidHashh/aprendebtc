# Checklist de testing responsive - aprendebtc.com

## Dispositivos a probar
- [ ] iPhone SE (375px)
- [ ] iPhone 14 Pro (393px)
- [ ] iPad (768px)
- [ ] Desktop (1280px)
- [ ] Desktop wide (1920px)

## Por cada dispositivo, verificar

### Landing page (/)
- [ ] Hero text legible, sin overflow
- [ ] Cards de niveles en grid correcto (1 col movil, 2 col tablet, 3 col desktop)
- [ ] Pilares visibles y clicables
- [ ] Footer completo

### Pagina de contenido (cualquier nivel)
- [ ] Sidebar oculta en movil, visible como menu hamburguesa
- [ ] Sidebar visible en desktop a la izquierda
- [ ] Breadcrumbs no desbordan
- [ ] Callouts legibles, no cortados
- [ ] Navegacion prev/next visible y clicable
- [ ] Ad slots no rompen el layout
- [ ] Texto maximo ~75 caracteres por linea (legibilidad)

### Herramientas
- [ ] Inputs ocupan el ancho completo en movil
- [ ] Outputs no desbordan (word-break funciona)
- [ ] Botones "Copiar" accesibles
- [ ] Grid de resultados se apila en movil (1 columna)
- [ ] Tablas con scroll horizontal en movil si son anchas

### Glosario
- [ ] Scroll fluido
- [ ] Terminos legibles
- [ ] Anchors de letras funcionan

### Header/Footer
- [ ] Menu hamburguesa funciona en movil
- [ ] Logo visible y enlaza a /
- [ ] Busqueda accesible
- [ ] Footer no cortado
