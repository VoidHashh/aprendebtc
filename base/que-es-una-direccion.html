¿Qué es una dirección Bitcoin?

El identificador donde recibes bitcoin. No es una cuenta bancaria. No tiene tu nombre. Y no deberías reutilizarla.

Definición

Una dirección Bitcoin es una cadena de caracteres alfanuméricos que sirve como destino para recibir bitcoin. Se genera a partir de tu clave pública mediante operaciones criptográficas.

Cuando alguien quiere enviarte bitcoin, le das una dirección. Los bitcoin enviados a esa dirección solo pueden gastarse con la clave privada correspondiente.

Tipos de direcciones

Existen varios tipos, identificables por cómo empiezan:

Legacy (P2PKH)

  • Empiezan por 1
  • Ejemplo: 1BvBMSEYstWetqTFn5Au4m4GFg7xJaNVN2
  • El formato original de Bitcoin
  • Fees más altas que formatos modernos

P2SH

  • Empiezan por 3
  • Ejemplo: 3J98t1WpEZ73CNmQviecrnyiWrnqRhWNLy
  • Usadas para SegWit envuelto y multisig
  • Compatibilidad intermedia

Native SegWit (P2WPKH)

  • Empiezan por bc1q
  • Ejemplo: bc1qar0srrr7xfkvy5l643lydnw9re59gtzzwf5mdq
  • El estándar actual recomendado
  • Fees significativamente más bajas

Taproot (P2TR)

  • Empiezan por bc1p
  • Ejemplo: bc1p5d7rjq7g6rdk2yhzks9smlaqtedr4dekq08ge8ztwac72sfr9rusxg3297
  • El formato más moderno (2021)
  • Mejor privacidad y eficiencia

Cómo se genera

Simplificado:

  1. 1. Tu wallet genera una clave privada (número aleatorio muy grande)
  2. 2. De la clave privada se deriva la clave pública (multiplicación en curva elíptica)
  3. 3. La clave pública se hashea (SHA256 + RIPEMD160)
  4. 4. Se añade información de versión y checksum
  5. 5. Se codifica en el formato correspondiente (Base58 o Bech32)

El resultado es una dirección que cualquiera puede usar para enviarte bitcoin, pero que solo tú (con la clave privada) puedes gastar.

No es una cuenta bancaria

Diferencias importantes:

No tiene nombre: La dirección no está vinculada a tu identidad en la red Bitcoin. Es pseudónima.

No tiene saldo fijo: Una dirección puede recibir múltiples transacciones de diferentes cantidades.

Puedes tener infinitas: Las wallets modernas (HD wallets) generan nuevas direcciones automáticamente.

Las transacciones son públicas: Cualquiera puede ver las transacciones de cualquier dirección en un explorador de bloques.

Por qué no reutilizar

Se recomienda usar una dirección nueva para cada transacción que recibes:

Privacidad

Si usas la misma dirección para todo, cualquiera que la conozca puede ver todo tu historial de transacciones y calcular tu saldo total.

Seguridad

Al gastar, revelas tu clave pública. Antes de gastar, solo el hash de la clave pública es público. Aunque teórico, usar direcciones una sola vez añade una capa de protección.

Las wallets lo hacen automático

Las HD wallets generan direcciones nuevas automáticamente. No tienes que hacer nada especial.

Verificar direcciones

Siempre verifica la dirección antes de enviar:

  • Copia/pega (no escribas manualmente)
  • Verifica los primeros y últimos caracteres
  • Si es una cantidad grande, envía una cantidad pequeña primero como prueba

El malware puede modificar direcciones en el portapapeles. Verificar previene pérdidas.
