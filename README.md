# aprendeBTC

`aprendebtc.com` es una web educativa multipágina sobre Bitcoin en español.

## Propósito

Ayudar a una persona sin experiencia a pasar de cero a niveles avanzados con una ruta clara y didáctica.

El proyecto se organiza en dos capas:

- `Pilares` = conocimiento de referencia.
- `Niveles` = acción paso a paso.

## Mapa del contenido

### Pilares transversales

- `/base/` Base de Conocimiento
- `/herramientas/` Herramientas interactivas
- `/comunidad/` Comunidad
- `/glosario.html` Glosario
- `/recursos.html` Recursos

### Niveles progresivos

- `/nivel-1/` Nocoinero Curioso
- `/nivel-2/` Ya Tengo Sats
- `/nivel-3/` Rabbit Hole
- `/nivel-4/` Down the Rabbit Hole
- `/nivel-5/` El Arquitecto
- `/nivel-6/` Satoshi
